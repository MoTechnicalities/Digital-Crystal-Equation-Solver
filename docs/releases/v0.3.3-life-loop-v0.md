# v0.3.3-life-loop-v0

Milestone:
- Milestone 3.3: Life Loop v0

Release date:
- Drafted on 2026-05-30

## Highlights

- Added Life Loop v0 core engine models and deterministic scheduler tick logic.
- Added Life Loop API routes:
  - `POST /v1/life-loop/tick`
  - `GET /v1/life-loop/state`
  - `GET /v1/life-loop/health`
- Added issue-ready JSON request/response examples for all three Life Loop routes.
- Added fixture-backed replay determinism gate for Life Loop state snapshots.

## Included Changes

### Engine
- Added `crates/digitalcrystal-engine/src/life_loop.rs`.
- Extended runtime configuration in `crates/digitalcrystal-engine/src/lib.rs` with:
  - `runtime.life_loop_state_path`
  - `runtime.life_loop_history_limit`

### API
- Integrated Life Loop state in `apps/api/src/main.rs`.
- Added persistence wiring for loop state save/load behavior.

### Conformance and Determinism

- Added deterministic replay test:
  - `crates/digitalcrystal-engine/tests/life_loop.rs::life_loop_tick_is_deterministic_from_replay_snapshot_fixture`
